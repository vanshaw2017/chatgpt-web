# chatgpt-web
## 项目介绍
本项目为简易聊天机器人项目，具有多轮聊天功能。</br>
底层调用chatgpt-3.5-turbo接口，本地运行时需要vpn,也可以直接将项目部署在海外服务器上，通过IP直接访问。
